# Cr0w (Static)
وێبسایتی مۆدێرن بۆ بڵاوکردنەوەی وێنە/ڤیدیۆ ڕۆژانە و هەوال.

## ✅ گرنگ
پێویستە فایلەکان **لە هەمان ئاستی index.html** بێت:
- /css
- /js
- /data
- /assets
- /media

ئەگەر تەنها index.html upload بکەیت، وێبسایتەکەت “سادە” دەردەکەوێت (وەک وێنەکەت).

## پۆست زیادکردن
1) فایلەکانت بخە `media/`
2) `data/posts.json` نوێ بکە
3) Push بکە → Cloudflare Pages خۆکار Deploy دەکات

## ڤیدیۆ
- mp4: `src` = `media/my-video.mp4`
- YouTube: `src` = `https://www.youtube.com/embed/VIDEO_ID`

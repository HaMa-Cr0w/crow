# Deploy on Cloudflare Pages (GitHub)
1) Repo دروست بکە و هەموو فایلەکان (نە فولدەری ناوەکی) لە root ـدا دابنێ.
2) Cloudflare → Workers & Pages → Create → Pages → Connect to Git
3) Framework preset: None
4) Build command: خالی
5) Output directory: /

✅ ئەگەر CSS کار ناکات: پێویستە /css و /js لە root بن.
